# Mintly 🍃

Mintly is a premium, beautifully designed personal finance tracker built with Flutter. It provides an intuitive and modern interface to help you manage your money, track expenses, create budgets, and achieve your financial goals with ease.

## 🚀 Features

*   **📊 Beautiful Dashboard:** Get a quick overview of your total balance, recent transactions, and spending progress at a glance with a stunning UI.
*   **💳 Accounts Management:** Track multiple accounts (cash, bank, credit cards) seamlessly.
*   **💸 Expense & Income Tracking:** Quickly log your transactions with categorized inputs and beautiful icons.
*   **📈 Advanced Reports:** Visualize your spending trends and category breakdowns with interactive pie and line charts.
*   **🎯 Savings Goals:** Set and track custom savings goals with animated progress bars.
*   **💰 Budgeting:** Create monthly budgets and keep your spending in check with dynamic warning indicators.
*   **📅 Calendar View:** See a day-by-day breakdown of your spending habits directly on an interactive calendar.
*   **🎨 Premium UI/UX:** Built using a modern glassmorphic and card-based design system that looks stunning in both light and dark modes.

## 🛠️ Tech Stack

*   **Framework:** [Flutter](https://flutter.dev/)
*   **State Management:** [Riverpod](https://riverpod.dev/)
*   **Routing:** [GoRouter](https://pub.dev/packages/go_router)
*   **Local Database:** [sqflite](https://pub.dev/packages/sqflite)
*   **Charts:** [fl_chart](https://pub.dev/packages/fl_chart)

## 📦 Getting Started

### Prerequisites
*   Flutter SDK (^3.10.0 or higher)
*   Dart SDK

### Installation

1.  **Clone the repository** (if applicable) or navigate to the project directory:
    ```bash
    cd mintly
    ```

2.  **Fetch dependencies**:
    ```bash
    flutter pub get
    ```

3.  **Run the app**:
    ```bash
    flutter run
    ```
